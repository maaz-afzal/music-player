# 🎵 Music Player

A simple music player built with HTML, CSS, and JavaScript. It plays songs from a local folder with basic controls — play, pause, previous, next, and a progress bar to jump to any part of the song.

---


## ✨ Features

- Play, pause, and skip tracks
- Progress bar to seek through songs
- Displays song title and artist name
- Clean and simple UI

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript

---

## 📁 How to Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/maaz-afzal/music-player.git
   ```

2. Go into the project folder:
   ```bash
   cd music-player
   ```

3. Open `index.html` in your browser — that's it!
